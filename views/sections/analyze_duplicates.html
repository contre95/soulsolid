How Duplicates Are Detected

Tracks are compared using Chromaprint fingerprints and Hamming distance — a bitwise comparison of the raw audio fingerprint. This means duplicates are detected by audio content, regardless of filename, tags, or format (MP3 vs FLAC).

  • ● Exact — same recording, possibly different encoding (e.g. MP3 + FLAC of the same master)
  • ● Fuzzy — likely the same song but slightly different (remaster, different cut, live version)
  • ● Pairs with duration differences over 10 s are skipped automatically
To also match by AcoustID recording ID, run the AcoustID analysis first so tracks have a recording ID assigned.
